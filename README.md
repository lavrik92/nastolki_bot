# Мой Telegram Bot

Бот для просмотра существующих настолок и голосования за ту, в которую играем сегодня

## Установка

1. Клонируй репозиторий
2. Создай виртуальное окружение
3. Установи зависимости: pip install -r requirements.txt
4. Создай файл .env на основе .env.example
5. Запусти: python src/main.py
